Sephardic Congregation of Cleveland Zmanim Page Usage

The zmanim calculation page pulls most of it's information from hebcal.com. By default it will use the location of the Sephardic Congregation of Cleveland, 2480 S. Green Road, University Heights, Ohio.

The easiest way to interact with the zmanim page is to use this form: Zmanim_Control_Form. There you will find options to view daily or Shabbat zmanim for specific dates, set a different location, and view detailed information on how these times are being calculated.

If you are more adventurous, you can control what information the page displays by adding various parameters to the URL line:

  • date=yyyy-mm-dd
    displays times for the specified day
  • shabbat=1
    will display times for the next upcoming Shabbat
    • NOTE: if you give a date AND use this parameter, and the date you give isn't actually shabbat, you'll get an error
  • zip=#####
    provides the times for the specified zipcode. Mutually exclusive from lat and long. Mutually exclusive from lat/long, city, or geoname.
  • geoname=######
    location specified by GeoNames.org numeric ID (See cities5000.zip from https://download.geonames.org/export/dump/.). Mutually exclusive from zip, city, or lat/long.
  • city=(cityname)
    location specified by one of the Hebcal.com legacy city identifiers (https://github.com/hebcal/dotcom/blob/master/hebcal.com/dist/cities2.txt). Mutually exclusive from zip, geoname, or lat/long.
  • lat=##.####
    latitude. Must also include longitude and tzid. Mutually exclusive from zip, city, or geoname.
  • long=##.####
    longitude. Must also include latitude and tzid. Mutually exclusive from zip, city, or geoname.
  • tzid=(timezone ID)
    TimezoneIdentifier (See List of tz database time zones: https://en.wikipedia.org/wiki/List_of_tz_database_time_zones). Mutually exclusive from zip, city, or geoname.
  • debug=1
    display all calculations and outputs for troubleshooting.

How to

To use those parameters, you start with the basic URL (https://clevelandsephardiminyan.com/wp-content/zmanim/sephardic_zmanim.php), then a question mark (?), and then add on from there. You can use parameters in any order. In between each parameter, you need to put an ampersand ("and sign", or "&")
